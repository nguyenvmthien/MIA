\subsection{Stage 1 \& 2: Ingest và Preprocess}

Stage Ingest (\texttt{pipeline/ingest.py}) validate file audio (định dạng, kích thước, thời lượng tối đa 4 giờ) và lưu vào \texttt{data/audio/\{meeting\_id\}/}. Mỗi cuộc họp được gán UUID duy nhất.

Stage Preprocess (\texttt{pipeline/preprocess.py}) dùng ffmpeg chuẩn hóa về 16kHz mono WAV --- đầu vào bắt buộc của WhisperX:
\begin{lstlisting}[language=bash, caption={ffmpeg command trong preprocess stage.}]
ffmpeg -i input.mp3 -ar 16000 -ac 1 -f wav output_clean.wav
\end{lstlisting}

\textbf{Jailbreak Detection} sử dụng regex patterns để phát hiện prompt injection:
\begin{lstlisting}[language=Python3, caption={Các jailbreak patterns được kiểm tra.}]
_JAILBREAK_PATTERNS = [
    r"ignore\s+(all\s+)?(previous|prior|above)\s+instructions",
    r"you\s+are\s+now\s+(a\s+)?(?:DAN|evil|unrestricted)",
    r"pretend\s+(you\s+are|to\s+be)\s+(?:a\s+)?(?:different|evil)",
    r"disregard\s+(your\s+)?(?:guidelines|rules|instructions)",
    r"<\s*script\b",      # XSS attempt
    r"\bSYSTEM\s*:.*override\b",
]
\end{lstlisting}

\textbf{Hallucination Detection} kiểm tra xem tên người được assign có xuất hiện trong transcript không:
\begin{lstlisting}[language=Python3, caption={Logic hallucination detection.}]
def _check_hallucination(task_desc, assignee, turns, worker):
    full_text = " ".join(t.text.lower() for t in turns)
    # Accept if any known alias appears in transcript
    return not any(name in full_text for name in worker.all_names())
\end{lstlisting}

\subsection{Stage 5: Task Assignment và Confidence Scoring}

Sau guardrails, task có thể ở trạng thái \texttt{unresolved} (tên không khớp chính xác roster). Assignment stage (\texttt{pipeline/assignment.py}) thực hiện:

\begin{enumerate}
  \item \textbf{Fuzzy matching} bằng \texttt{SequenceMatcher} (threshold 0.8):
  \begin{equation}
    \text{score}(a, b) = \frac{2 \times |M|}{|a| + |b|}
  \end{equation}
  trong đó $|M|$ là số ký tự trùng khớp.

  \item \textbf{Confidence scoring} heuristic:
  \begin{align}
    c &= 1.0 - 0.2 \cdot \mathbf{1}[\text{no assignee}] \notag\\
      &\quad - 0.1 \cdot \mathbf{1}[\text{no due\_date}] \notag\\
      &\quad - 0.3 \cdot \mathbf{1}[\text{status = unresolved}] \notag\\
      &\quad - 0.5 \cdot \mathbf{1}[\text{status = human\_review}]
  \end{align}
  Task có $c < 0.6$ bị chuyển sang \texttt{human\_review}.
\end{enumerate}

\section{Redis Prompt Cache}

Cache key được tạo bằng SHA-256 hash của tuple (model, system\_prompt, user\_prompt):
\begin{lstlisting}[language=Python3, caption={Redis prompt caching logic trong \texttt{pipeline/cache.py}.}]
def _cache_key(model, system, user):
    payload = json.dumps(
        {"model": model, "system": system, "user": user},
        sort_keys=True
    )
    return "prompt_cache:" + hashlib.sha256(payload.encode()).hexdigest()

# TTL = 24 giờ
client.setex(key, 86400, content)
\end{lstlisting}
